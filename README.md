## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CedricMlg/P6_FishEye_Malingre_Cedric
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Launch project
   ```js
   npm run dev
   ```
4. Open project 
   <br />
   Click on the local link while pressing CTRL
