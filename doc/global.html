(async) init()

The init function is an async function that calls the getPhotographers function, which returns a promise that resolves to an object with two properties, photographers and media. The init function then calls the displayData function, passing in the photographers property of the object returned by the getPhotographers function, and then calls the displayMedia function, passing in the media property of the object returned by the getPhotographers function.
